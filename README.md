## Requirement

Swift 4.0

## Installation

- Swift Package Manager
- Cocoapods: `pod 'Pangu-Swift', '~> 1.0'`

## Useage

```swift
import Pangu_Swift

print( "Mr.龍島主道：「Let's Party!各位高明博雅君子！」".spaced )
// Mr. 龍島主道：「Let's Party! 各位高明博雅君子！」
```
